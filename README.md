## Quick start

1) Install Python dependencies (recommended in a virtual environment):

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

2) (Optional) Regenerate the JSON files used by the dashboard:

```bash
# generate JSON files (reads Student_Performance_Data.csv)
python generate_viz_data.py
```

3) Open the dashboard

- Option A: Open `index.html` directly in a browser (works for local files), or
- Option B: run a simple static server and visit `http://localhost:8000`:

```bash
python -m http.server 8000
```

Reload the page after running `generate_viz_data.py` to see embedding/SHAP visuals if generated.

## Data & models

- The included CSV (`Student_Performance_Data.csv`) is the raw dataset used for EDA and model training.
- `generate_viz_data.py` performs preprocessing, trains or loads models (if present), computes embeddings and SHAP values (when dependencies are installed), and writes JSON files to `data/` consumed by the frontend.
- Trained models (if produced) are saved into `models/`.

If you need SHAP/UMAP embedding support, install the optional packages listed in `requirements.txt`.

## Usage notes

- The dashboard is intended for exploration and teaching. The prediction tool is for demonstration; production deployments should run model inference on a secure backend.
- The provided JSON files include sample outputs so the UI can be used without running the Python scripts.
