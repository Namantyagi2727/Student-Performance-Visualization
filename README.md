# Student Performance Visualization

A concise, human-friendly dashboard for exploring student performance data and viewing results from machine learning models trained to predict semester GPA.

This repository contains a static frontend (`index.html`, `app.js`, `styles.css`) that reads example JSON files in `data/`, plus Python scripts and notebooks used to prepare the data and (optionally) train models.

## What this project does

- Provides interactive charts for EDA: distributions, correlations and categorical breakdowns.
- Compares model performance (RÂ², RMSE, MAE) across several regressors.
- Offers a small prediction demo where you can adjust student features and see a predicted GPA.
- Shows a student-level embedding plot (UMAP / t-SNE) when embedding data is available.
- Optionally displays SHAP summaries if they are generated by the Python pipeline.

## Quick start

1. (Optional but recommended) Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

2. (Optional) Regenerate the JSON files used by the dashboard:

```bash
# reads Student_Performance_Data.csv and writes JSON outputs to data/
python generate_viz_data.py
```

3. Open the dashboard

- Open `index.html` directly in a browser, or serve files locally:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```

After running the generator script, reload the page to pick up any new embedding or SHAP files.

## Data and models (notes)

- `Student_Performance_Data.csv` is the raw dataset used for analysis and training.
- `generate_viz_data.py` handles preprocessing and exports JSON files into `data/`. It may also train models and compute embeddings/SHAP values if the required packages are installed.
- Saved models (if created) are placed in `models/`.

If the optional packages for SHAP/UMAP are not installed, the script will skip those steps and leave example JSON in `data/` for the frontend to use.
